# Publishing Playbook

Bookmark Maxxing issues should feel useful, not like link dumps.

## Title Formula

Use a transformation:

```text
Bookmark Maxxing #1: Turning Bookmarks into Skills
```

Other examples:

- Bookmark Maxxing #2: Turning Saved Posts into Agent Loops
- Bookmark Maxxing #3: Turning GitHub Stars into Build Ideas
- Bookmark Maxxing #4: Turning Newsletter Links into Operating Principles

## Opening Formula

```text
Most people collect bookmarks.

Builders should be collecting skills.

This week I analyzed [source] and found [pattern].

The goal was not another recap.

It was to answer:

Which of these ideas are worth turning into reusable skills?
```

## Body Formula

Each section should use:

```text
## Theme: [Theme]

**Skill: [Reusable behavior.]**

[What the sources said.]

[Why it matters.]

[How builders can use it.]
```

## Ending Formula

```text
We already save the best ideas. The problem is not discovery. It is conversion.

Bookmark Maxxing is about turning saved ideas into durable systems.

Bookmarks are the input. Skills are the output.
```

## Source Etiquette

- Attribute people by name and link.
- Do not imply endorsement.
- Tag only a small number of highly relevant people when sharing.
- Prefer source links in the article and a short attribution list in the share post.
