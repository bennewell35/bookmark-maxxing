# Support

Bookmark Maxxing is an open-source framework.

## Best Places to Ask

- Use GitHub issues for framework bugs, unclear docs, template problems, or example requests.
- Use pull requests for concrete improvements.
- Use security reporting for private security concerns.

## Not Support

This project does not provide support for:

- recovering social media accounts
- bypassing login, paywalls, rate limits, or platform restrictions
- scraping private data
- automating likes, reposts, replies, follows, or unbookmarking
- debugging unrelated AI tools or browser extensions

## Good Support Requests

Good requests include:

- what source you are trying to analyze
- what output you expected
- what prompt/template you used
- what result was confusing or unsafe
- whether the source data is public or private

Do not paste private bookmark exports, tokens, cookies, passwords, or account data into public issues.
